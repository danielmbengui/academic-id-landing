# MINSEC — Digital Transformation, Blockchain & Sovereignty

![MINSEC](./public/logo.png)

## Overview

MINSEC is a technology company specialized in digital transformation, blockchain infrastructure, cybersecurity, smart contracts, decentralized finance (DeFi), Web3 solutions, and responsible cryptocurrency mining.

Our mission is to help governments, public institutions, private organizations, and strategic partners accelerate their digital transition through secure, transparent, and scalable technologies.

The company actively contributes to the development of sovereign digital infrastructures, blockchain adoption, and innovative economic models across Africa.

---

## Vision

To become a leading African technology partner for digital sovereignty, blockchain innovation, and next-generation digital infrastructure.

MINSEC believes that blockchain is not only a technology but a foundation for transparency, trust, efficiency, and sustainable economic growth.

## Installation

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

Build production:

```bash
npm run build
```

Start production:

```bash
npm start
```
